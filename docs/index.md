## Installation
PSO365 is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/PSO365/).

To Inspect:
```powershell
Save-Module -Name PSO365 -Path <path>
```
To install:
```powershell
Install-Module -Name PSO365 -Scope CurrentUser
```
